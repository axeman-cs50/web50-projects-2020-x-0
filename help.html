Viewing a gallery

To view a gallery, select the "view gallery" button from the dashboard. You will be able to select among any galleries you have created. Your gallery items will be displayed in a compressible grid format. From here you simply click on the item you would like to view and it will open with the proper viewer (i.e. Photos, MS Word, Media Player, etc.)

Creating a gallery

To create a gallery, select the "create gallery" button from the dashboard. You will be taken to the create a gallery screen where you will be prompted to enter a name for the new gallery. After providing a name, you will be able to click the "add item to gallery" button, which will open a file browser from which you can select the new file. After selecting the file, click on "open" in the dialog to upload the item to the folder. You may repeat this process as many times a necessary to complete your gallery.

Types of Media

Any type of media that your computer recognizes should be able to be uploaded to a gallery. These include:
Image Types:
  1. jpeg
  2. jpg
  3. png
  4. tiff
  5. bmp
Other file types:
  • mp3
  • wav
  • mov
  • txt
  • doc
  • rtf
  • ppt
  • xlxs
And virtually any other type of file.
